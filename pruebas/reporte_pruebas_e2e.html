4. Pruebas E2E (End to End) con Selenium IDE

Objetivo: Automatizar el comportamiento real del usuario desde el navegador, simulando flujos completos de uso del sistema.

Descripción de las Pruebas Realizadas

E2E-01: Verificación de Carga de Página
Descripción: Se verifica que la página principal del sistema cargue correctamente en el navegador.
Herramienta: Selenium WebDriver
Resultado: ✓ La página carga sin errores y muestra el título correcto.

E2E-02: Login de Paciente
Descripción: El paciente inicia sesión en el sistema con sus credenciales.
Flujo: 1. Acceder a la página de login 2. Ingresar email y contraseña 3. Presionar botón de iniciar sesión 4. Verificar acceso al dashboard
Resultado: ✓ El formulario de login está disponible y funcional.
Credenciales de prueba: paciente@test.com / paciente123
E2E-03: Agendar Cita Médica
Descripción: El paciente agenda una nueva cita médica.
Flujo: 1. Iniciar sesión como paciente 2. Ir a sección "Nueva Cita" 3. Seleccionar médico 4. Seleccionar fecha y hora disponible 5. Seleccionar tipo de consulta 6. Confirmar solicitud
Resultado: ✓ El sistema permite agendar citas en horarios disponibles.
Validaciones: Se verifica que no permita agendar en horarios ocupados.
E2E-04: Visualizar Historial de Citas
Descripción: El paciente visualiza su historial de citas médicas.
Flujo: 1. Iniciar sesión como paciente 2. Ir a sección "Mis Citas" o "Historial" 3. Ver lista de citas (pasadas y futuras)
Resultado: ✓ El paciente puede ver todas sus citas.
Información mostrada: Fecha, hora, médico, estado, tipo de consulta.
E2E-05: Cancelar Cita
Descripción: El paciente cancela una cita programada.
Flujo: 1. Iniciar sesión como paciente 2. Ir aMis Citas 3. Seleccionar cita a cancelar 4. Presionar botón "Cancelar" 5. Confirmar cancelación
Resultado: ✓ La cita se marca como cancelada.
Validación: Solo citas pendientes oconfirmadas pueden cancelarse.
E2E-06: Login de Médico
Descripción: El médico inicia sesión en el sistema.
Flujo: 1. Acceder a la página de login 2. Ingresar credenciales de médico 3. Verificar acceso al panel de médico
Resultado: ✓ El médico puede iniciar sesión.
Credenciales de prueba: medico@test.com / medico123
E2E-07: Ver Agenda del Médico
Descripción: El médico visualiza su agenda de citas.
Flujo: 1. Iniciar sesión como médico 2. Ir a sección "Mi Agenda" 3. Ver citas del día
Resultado: ✓ El médico puede ver todas sus citas programadas.
Información mostrada: Paciente, fecha, hora, tipo de consulta.
E2E-08: Confirmar/Cancelar Cita por Médico
Descripción: El médico confirma o cancela una cita pendiente.
Flujo: 1. Iniciar sesión como médico 2. Ver citas pendientes 3. Seleccionar cita 4. Elegir "Confirmar" o "Cancelar"
Resultado: ✓ El médico puede gestionar el estado de las citas.
Estados disponibles: pendiente, confirmada, cancelada, completada.
